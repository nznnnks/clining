# Клининговая компания - React Frontend

Веб-интерфейс для клининговой компании, разработанный на React с темной темой.

## Структура проекта

```
clining_project/
├── public/
│   └── index.html
├── src/
│   ├── components/
│   │   ├── Header.js          # Шапка сайта с навигацией
│   │   ├── Hero.js            # Главный баннер
│   │   ├── About.js           # Раздел "О нас"
│   │   ├── Portfolio.js       # Портфолио работ до/после
│   │   ├── Packages.js        # Готовые пакеты услуг
│   │   ├── WorkSteps.js       # Этапы работы
│   │   ├── Calculator.js      # Калькулятор стоимости
│   │   ├── ContactButtons.js  # Плавающие кнопки связи
│   │   └── Footer.js          # Подвал сайта
│   ├── styles/
│   │   └── dark-theme.css     # Темная тема
│   ├── App.js
│   ├── App.css
│   └── index.js
├── package.json
└── README.md
```

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите проект в режиме разработки:
```bash
npm start
```

3. Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Сборка для продакшена

```bash
npm run build
```

## Основные функции

- ✅ Главная страница с баннером
- ✅ Раздел "О нас"
- ✅ Портфолио с работами до/после
- ✅ Готовые пакеты предложений
- ✅ Этапы работы
- ✅ Калькулятор расчета стоимости
- ✅ Функциональные кнопки звонка и связи через мессенджеры
- ✅ Темная тема оформления
- ✅ Адаптивный дизайн

## Технологии

- React 18.2.0
- CSS3 с CSS переменными для темной темы
- Адаптивная верстка

## Контакты

Для изменения контактных данных отредактируйте файлы:
- `src/components/Header.js` - телефон в шапке
- `src/components/ContactButtons.js` - кнопки связи
- `src/components/Footer.js` - контакты в подвале
